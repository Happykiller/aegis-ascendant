# BRIEF-0013 — Planche concept de l'Aegis Citadel

- **Statut** : assigné
- **Assigné à** : asset-forge
- **Rédigé par** : concepteur intérimaire (Codex, continuité exceptionnelle)
- **Date** : 2026-07-11

## Objectif

Créer une planche concept originale de l'Aegis Citadel, forteresse mobile alliée et second mode
de contrôle du prototype.

## Silhouette canonique

- noyau central prismatique très lisible ;
- exactement deux bras-batteries massifs mais de détails asymétriques ;
- grande baie d'appontage sous le noyau ;
- corps axial court et large, sans silhouette humanoïde ;
- surfaces segmentées, lumières d'échelle et éléments mécaniques animables ;
- lecture immédiate à très grande distance.

## Systèmes visibles

- Point Defense Grid réparti sur la coque ;
- Twin Rail Batteries sur les deux bras ;
- Aegis Shield émis depuis trois nœuds secondaires ;
- Helios Lance intégrée dans l'axe prismatique ;
- batterie externe défendable ;
- baie d'appontage et transfert de commande ;
- zones de dommages localisées sans perdre la silhouette.

## États requis

1. veille ;
2. bouclier actif ;
3. batteries déployées ;
4. charge Helios Lance ;
5. dégâts lourds avec un bras endommagé.

## Direction artistique

Helios Vanguard : blanc cassé `#EDEAE3`, bleu profond `#1C2B5E`, cyan `#3FD9E8`, or
`#E4B54A`, rouge sécurité `#C93A31` limité, structure anthracite `#24252B`. Architecture
militaire modulaire, céramique technique, noyau énergétique facetté et lumières d'échelle.

## Composition

- grande vue de dessus ;
- vues de face, profil, arrière et oblique ;
- cinq miniatures d'état ;
- détails du noyau, de la baie, des rails, d'un nœud de bouclier et du Point Defense Grid ;
- silhouette comparative avec Aurora Spear et Specter-9, sans dimensions chiffrées ;
- fond technique bleu-noir, sans texte.

## Contraintes IP

Création entièrement originale. Aucun nom, logo, classe, marquage ou design de licence. Ne pas
reprendre les forteresses présentes dans les références internes. Aucun torse, bras humanoïdes,
jambes, tête, transformation ou silhouette de navire maritime.
