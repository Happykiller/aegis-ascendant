# Prompt de génération — Familles du Null Choir

- **Brief** : `docs/forge/briefs/BRIEF-0007-enemy-families-concept.md`
- **Outil prévu** : imagegen intégré
- **Type** : stylized-concept

## Prompt

Créer une planche de concept art sans texte contenant exactement sept familles de vaisseaux
ennemis biomécaniques originaux, chacune isolée dans une cellule : petit scout axial en forme
d'éclat, intercepteur latéral en arc brisé asymétrique, mine compacte à segments radiaux, drone
de contact à trois pinces courtes, tourelle ancrée avec base mécanique, bombardier ventru portant
des modules de mines, transporteur de bouclier large autour d'un projecteur central. Pour chaque
famille, montrer une vue de dessus dominante et une petite vue oblique. Hiérarchie d'échelle
claire. Carapaces anthracite, segments violet sombre, inserts ivoire froid, fissures et noyaux
magenta, vert maladif très limité. Esthétique biomécanique militaire stylisée, silhouettes
asymétriques, dessin industriel précis sur fond technique bleu-noir.

Éviter absolument : texte, lettres, nombres, labels, logos, emblèmes, filigrane, licence,
marque, artiste, formes humanoïdes, visages, insectes réalistes, symétrie parfaite généralisée,
ailes d'avion conventionnelles, designs identiques entre cellules, bataille ou décor narratif.
