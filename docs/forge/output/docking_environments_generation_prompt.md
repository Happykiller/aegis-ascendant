# Prompt de génération — Environnements d'appontage

- **Brief** : `docs/forge/briefs/BRIEF-0009-docking-environments.md`
- **Outil prévu** : imagegen intégré
- **Type** : stylized-concept

## Prompt

Créer une planche concept sans texte montrant cinq étapes d'une séquence d'appontage militaire
spatiale originale : grande baie de lancement ouverte vers les étoiles, catapulte axiale avec
zone de préparation, tunnel d'approche large, rail magnétique d'arrimage, mécanisme final
d'accrochage dorsal autour d'un petit chasseur triangulaire. Pour chaque étape, associer une
perspective basse immersive et une petite vue de dessus lisible. Ajouter une bande de détails :
portes blindées segmentées, balises cyan/or/rouge, panneaux sans texte, bras d'assistance,
conduits d'énergie et marquages directionnels géométriques. Architecture blanc cassé, bleu
profond, anthracite, lumière cyan, accents or et rouge sécurité limité. Centre des corridors
dégagé, détails mécaniques concentrés sur les bords, fond technique bleu-noir entre cellules.

Éviter absolument : texte, lettres, nombres, labels, logo, emblème, filigrane, licence, marque,
artiste, architecture reconnaissable d'une œuvre existante, robot humanoïde, foule, bataille,
couloir étroit illisible, surcharge lumineuse au centre de la trajectoire.
